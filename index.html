Generalised Dot Maps

Notes

On zoom and/or pan, the underlying map is zoomed with svg zoom (faster than reprojecting). Then, the projection is updated and used to display the updated point collection. The dataset used here is a list of all Starbucks stores worldwide (Source). I modified it to only contain unique locations (removing all stores whose location is already occupied by another store), since coincident locations (currently) lead to errors.

Options

Updates on next zoom or pan.

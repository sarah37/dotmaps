Generalised Dot Maps

Notes:

The grey box shows the bounding box, the dataset is filtered for dots inside this box so that calculations on the dots are only done on the visible ones.
Currently, this simply plots all dots, which is why it is slow at small scales. When zooming in, there are fewer dots and it gets smoother.
On zoom and/or pan, the underlying map is zoomed with svg zoom (faster than reprojecting). Then, the projection is updated and the dots are reprojected.
